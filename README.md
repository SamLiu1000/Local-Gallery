# Local Gallery

> A lightweight yet powerful image management tool with AI metadata support —
> a hub of art and inspiration you'll want to open every day.

---

## Features

1. **Smart thumbnail caching** — load once, browse forever with silky-smooth performance.

2. **Fast & solid search** — find any image instantly across your entire collection.

3. **Rich tagging system** — plain text tags, avatar-style tags, and full HTML tags. Supports SVG icons, system font icons, and lets you turn folders into tags for easier organization.

4. **AI integration** — customize API endpoints and system prompts to auto-generate image descriptions.

5. **Multiple prompts per image** — store several prompt descriptions for a single image, either via API or manually.

6. **AI-generated image metadata** — supports reading metadata from nearly all mainstream AI image generation tools.
