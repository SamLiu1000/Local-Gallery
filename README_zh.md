# Local Gallery

> 一款轻量却强大的图片管理工具，支持 AI 元数据 ——
> 一个用于管理艺术图片与灵感、让你每天都想打开浏览的相册。

---

## 功能特性

1. **智能缩略图缓存** — 加载一次，永久流畅浏览，体验丝滑。

3. **快速稳定的搜索** — 在整个图库中即时定位任意图片。

4. **丰富的标签系统** — 支持纯文本标签、头像式标签和完整 HTML 标签。兼容 SVG 图标、系统字体图标，并可将文件夹转换为标签，让整理更轻松。

   你可以创建纯文本标签、图文标签、纯图标标签、图片+文字标签、纯图片标签，甚至 HTML 特效标签。  
   这意味着只要你的想象力足够丰富，标签样式可以变成任何你想要的形态。  
   别忘了——你还可以将文件夹路径转换为标签，并随心定制它的外观。

<div align="center">
<video src="https://github.com/user-attachments/assets/a517f01e-b4b1-4456-955d-dd27f8c16060" width="360" controls></video>
</div>

5. **AI 集成** — 自定义 API 端点和系统提示词，自动为图片生成描述。

6. **多提示词支持** — 为单张图片存储多条提示词描述，可通过 API 或手动填写。

   这是本程序的核心功能之一。我通常用它来搭建自己的灵感库。  
   你可以自定义 API 的系统提示词，让 AI 为你点评或描述作品；  
   也可以手动填写提示词，记录你自己对这幅作品的理解。同时支持批量 AI 描述。  
   点击相册顶部的编辑按钮，左侧文件夹导航下方便会出现编辑选项。
